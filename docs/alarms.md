Alarm bits are set when alarms occurs. The bits are:

  | Bit |  Description |
  |----- |-------------|
  | 0   |  Reserved. |
  | 1   |  Low alarm has occurred. |
  | 2   |  High alarm has occurred. |
  | 3   |  Reserved |
  | 4   |  Reserved |
  | 5   |  Reserved |
  | 6   |  Reserved |
  | 7   |  Reserved |

Remember that alarm bits are cleared when read.
