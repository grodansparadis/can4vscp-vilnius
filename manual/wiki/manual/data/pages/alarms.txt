====== Alarms ======

^ Bit ^Description ^
| 0 | Protection timer elapsed. |
| 1 | An input bit has gone high and the alarm condition for it is enabled. |
| 2 | An input bit has gone low and the alarm condition for it is enabled. |
| 3 | Reserved |
| 4 | Reserved |
| 5 | Reserved |
| 6 | Reserved  |
| 7 | Reserved  |

Remember that alarm bits are cleared when read.
